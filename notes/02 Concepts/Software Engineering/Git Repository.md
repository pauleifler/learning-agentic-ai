# Git Repository

## Definition

A Git repository is a directory that Git tracks for version control. It is identified by a hidden `.git` folder.

## Why it matters

A project should normally have **one** Git repository at its root.

Nested repositories are uncommon and are usually only used for Git submodules.

## How it works

- Identify a git repository using `ls -la`
