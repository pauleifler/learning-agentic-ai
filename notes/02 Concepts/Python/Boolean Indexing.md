# Boolean Indexing

## Definition

Boolean indexing filters a pandas DataFrame using conditions that evaluate to `True` or `False`.

## Why it matters

This concept will come up constantly in the next few days as we build the Football Analytics Copilot. Understanding it well will make the rest of the project much easier.

## How it works

- `==` Equal
- `!=` Not equal
- `>` Greater than
- `<` Less than
- `&` AND
- `|` OR

## Example

```python

data[data["league"] == "Premier League"]

```
