# Feature Engineering

## Definition

Feature engineering is the process of creating new variables or metrics from raw data that better describe the underlying behaviour.

## Why it matters

LLMs reason better when given meaningful features rather than raw records.

## Example

Derived features
- Goal difference
- Points

- Rolling form
- BTTS rate
